關於Package Control

＊　基本上，Package Control 是把功能給整合到Command Palette之中，然後利用「Package」這個關鍵字來帶出相關的Package Control項目然後讓Sublime 3去執行指令。

＊　安裝後可去管理這些外掛 Install Package(安裝)、List Package(已安裝列表)、Remove Package(移除) 與 Upgrade Package(升級)

安裝步驟

Step 1，在 chrome 瀏覽器中搜尋 "package control install"，找到 Package Control 的安裝網頁：https://packagecontrol.io/installation

Step 2，將其中 for SUBLIME TEXT 3 的一段 Python 程式複製下來：

import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

Step 3，開啟 Sublime Text 3，按＜Ctrl-～＞，在最底下的 console 中，按＜Enter＞執行。執行後會出現一個錯誤提示，結束安裝，不必理它。

Step 4，接著按＜Ctrl-Shift-P＞呼叫出 Package Control，在視窗中輸入＂PCI＂，搜尋＂Package Control: Install Package＂，點選安裝。

Step 5，同樣的方法，接著安裝 "htmlcssjs" 及 ＂emmet＂

Step 6，修改 node.js 的環境變數。

　　　　點選 Preference \ Package Setting \ HTML/CSS/Js \ Set 'node' Path，在視窗中第六行， "windows" 後的值改為 "node"。第 12 行,"format_on_save"　的值改為"true"。

基本上開發環境的建立工作即大功告成！
